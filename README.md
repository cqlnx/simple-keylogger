## What is this?

This is a very simple Python keylogger created as a learning project.  
It captures keyboard input and sends it to a Discord webhook.

---

## How to test it (your own system only)

To test this script only on your own computer:

1. run pip install -r requirements.txt
2. Create a Discord webhook  
3. Open the Python script  
4. Replace the webhook placeholder  
   webhook_url = "your webhook goes here"  
5. Run the script locally  

No additional setup is required.
